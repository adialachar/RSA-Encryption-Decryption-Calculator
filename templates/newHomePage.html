How does it work

p, q, n, phiN, d, e There are two parts to RSA Encryption: the Public Key and the Private key The Public Key is used to encrypt information, while the private key is used to decrypt information. The information necessary to build the public key is usually publically available , while the private key is difficult to calculate and cannot be found easily. Before we get into the weeds, lets define all of the numbers involved in RSA Encryption and Decryption p: a prime number q: a prime number that is different from p n: the product of p and q phiN: the product of (p -1) and (q - 1) e: A number that is co-prime to phiN d: The Private Key c: The variable usually used to represent the encrypted number m: The variable usually used to represend the un-encrypted or decrypted number Now that we know all of the numbers involved, we can start defining what the Public Key and Private Key are The Public Key is made up of two parts: n and e, usually written as (n, e) The Private Key is just made up of one number: d d is a number that requires 3 other numbers to solve for. These numbers are p,q, and e. To encrypt information, the process is as follows: Take the information from the public key (n, e) and take the number you want to encrypt and plug it into this formula The formula to encrypt is c = m^e mod n To decrypt information, the process is as follows: If we aren't given p or q, we need to crack the public key in order to decrypt the information. n is the product of p and q, both of which are prime numbers, so in order to get these numbers we need to crack n. This is very difficult, and is the main reason why RSA encryption is so effective. Because p and q are prime, the only factors of n are p , q and 1, making it extreemely difficult to find those specific numbers. After getting p and q, we need to calculate d. This is done using the formuala d e triple bar 1 mod phiN. phiN is calculated by takking the product of (p - 1) * (q - 1) after calculating d we can use the following formula to decrypt information: The formula to decrypt is m = c^d mod n

Where is it used today

RSA is used in a variety of systems that we use every day. Some of the services that use RSA encryption are web browsers, chat applications email clients, VPNs among many other services. RSA is one of hte toughest encryptions to crack because it takes advantage of unique properties that prime numbers have, and this toughness makes it used in many applications.
